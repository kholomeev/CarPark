# Краткое описание #
Приложение "Автостоянка" является отличным решением, если вы собираетесь использовать её для формирования и мониторинга мест в автостоянке. В этой программе будут реализованы функции регистрации пользователей и сотрудников, а также обеспечение взаимодействия между ними.

# Функции #
Если вы являетесь оператором, вы можете осуществлять следующие функции:
* Составлять и выдавать билеты пользователям;
* Делать напоминания о задолженности;

Если вы менеджер базы данных, то вы сможете:
* Управлять данными о пользователях;
* Управлять данными об автомобилях;
* Управлять данными о местах;
* Импортировать существующие БД в виде запросов SQL в программу;
* Экспортировать существующие БД из программы в виде запроса SQL;

# Установка #
## Способ установки 1: через консоль (терминал) ##
Вам всего лишь нужно клонировать данный репозиторий себе в компьютер.
1. Установите [Git](https://git-scm.com/).
2. Выберите место, куда вы хотите клонировать репозиторий.
3. Откройте терминал и напишите следующее: `git clone https://github.com/kholomeev/CarPark.git`.

## Способ установки 2: непосредственно на сайте ##
Вам нужно будет скачать его в формате .zip.
Процесс скачивания указан на фотографии ниже.

На данный момент программу можно запустить только в Visual Studio. Просто клонируйте репозиторий через эту программу и запускайте.
